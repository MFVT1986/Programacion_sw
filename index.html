ETIQUETAS HTML


ETIQUETA DESCRIPCIÓN
" !doctype html " Tipo de documento (versión de html empleada)
" html " Representa la raíz de un documento HTML o XHTML. Todos los
demás elementos deben ser descendientes de este elemento.
" head " Delimita el encabezado del documento.
" title " Título del documento (se muestra en la pestaña del navegador).
" header " Define la cabecera de una página o sección. Usualmente contiene un
logotipo, el título del sitio Web y una tabla de navegación de contenidos.
" footer " Define el pie de una página o sección. Usualmente contiene un mensaje
de derechos de autoría, algunos enlaces a información legal o
direcciones para dar información de retroalimentación.
" br "> Agrega un salto de linea.
" style " Hoja de estilo incluida en el documento.
" div " Agrega una división en el documento.
" body " Delimita el cuerpo del documento.
